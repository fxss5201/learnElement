# 项目说明

运行本项目，需要将 VuePress 安装为本地依赖：

# 将本项目克隆到本地
git clone git@github.com:fxss5201/learnElement.git
# git clone https://github.com/fxss5201/learnElement.git

# 安装依赖
npm install

# 开始写作
npm run docs:dev

# 构建
npm run docs:build
